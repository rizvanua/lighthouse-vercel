Lighthouse API

This is a serverless API for running Lighthouse audits on websites.

Available Endpoints

Health Check

GET /api/health

Analyze URLs

POST /api/analyze
Content-Type: application/json

{
  "urls": ["https://example.com"]
}

Example Usage

curl -X POST \
  https://lighthouse-vercel-pink.vercel.app/api/analyze \
  -H 'Content-Type: application/json' \
  -d '{
    "urls": ["https://example.com"]
  }'
